# Sokoban

The classic "warehouse keeper" game from Japan, now for your terminal.

Written for fun, code is not clean or good. Don't judge me, please! :)

![Sokoban Screenshot](https://i.imgur.com/ZAzy8Nn.png)

Font: [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P)

Levels taken from [Math is Fun](https://www.mathsisfun.com/games/sokoban.html).

## Controls

- **w,s,a,d**: Movement
- **r**: Reset level
- **n,p**: Next and previous levels
- **Arrow keys**: Move the map
